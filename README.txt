 📉 Customer Churn Prediction

A machine learning project to predict whether a customer will churn, based on historical behavior and attributes. Built using Python (Jupyter Notebook), SQL, and Power BI.

---

🔍 Problem Statement

Businesses lose revenue when customers leave — known as "churn". The goal is to **predict churn early** so companies can take action and retain valuable customers.

---

🧰 Tools & Technologies

 **Python** (Pandas, Scikit-learn, Matplotlib)

 **Jupyter Notebook**

 **SQL** (for data extraction and transformation)

 **Power BI** (for dashboard and data visualization)

 **Git & GitHub** (for version control and collaboration)

---

📊 Project Components


churn_model.ipynb: Contains full EDA, feature engineering, and machine learning model (e.g., logistic regression or random forest) 

customer_churn.sql: SQL queries used to clean and prepare the dataset 

dashboard.pbix: Interactive Power BI dashboard for executive-level churn insights |

---

🚀 Key Features

- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Feature selection & encoding
- ML model training and evaluation
- Interactive dashboard with churn KPIs
